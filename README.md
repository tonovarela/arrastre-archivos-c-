# arrastre-archivos

Proyecto en C# para gestionar el arrastre y procesamiento de archivos HTML.

## Descripción
Esta aplicación permite mover y organizar archivos HTML desde una carpeta de origen a una carpeta de destino, facilitando la gestión de archivos de manera automatizada.

## Estructura del proyecto
- `archivos/`: Carpeta de archivos HTML de origen.
- `destino/`: Carpeta de destino para los archivos procesados.
- `clases/`: Clases auxiliares del proyecto.
- `exceptions/`: Definición de excepciones personalizadas.
- `Program.cs`: Punto de entrada de la aplicación.

## Requisitos
- .NET 8.0 o superior
- macOS, Windows o Linux

## Uso
1. Clona el repositorio:
   ```sh
   git clone https://github.com/tonovarela/arrastre-archivos-c-.git
   cd arrastre-archivos-c-
   ```
2. Compila el proyecto:
   ```sh
   dotnet build
   ```
3. Ejecuta la aplicación:
   ```sh
   dotnet run
   ```
